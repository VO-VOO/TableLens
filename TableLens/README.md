# TableLens

Native macOS menu bar app for Baidu table OCR.

## Structure
- `TableLens/App.swift`: main app source
- `TableLens/Info.plist`: app metadata
- `TableLens.xcodeproj`: Xcode project
- `build_release.sh`: local build/export script

## Build
```bash
./build_release.sh
```

Exports the app to `/Applications/TableLens.app`.
