# TableLens

TableLens 是一个 macOS 菜单栏表格识别工具。

## 功能
- 全局热键唤起截图
- 截图框可继续调整后再确认识别
- 调用百度表格识别 API 导出 Excel
- 成功/失败会发送 macOS 原生通知
- API Key / Secret Key 本地加密保存

## 使用方式
1. 打开 App，先在设置中填写百度 API Key 和 Secret Key。
2. 选择 Excel 保存目录。
3. 使用全局热键唤起截图，框选表格区域后按回车识别。
4. 成功后会在保存目录中生成 Excel。
